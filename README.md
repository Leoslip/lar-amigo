# Site Lar Amigo

Site para ONG fictícia de adoção de animais.

## Arquivos

- `index.html` - Página inicial
- `sobre.html` - Sobre a ONG  
- `contato.html` - Formulários
- `style.css` - Estilos

## Como usar

Coloque todos os arquivos na mesma pasta e abra `index.html`.
